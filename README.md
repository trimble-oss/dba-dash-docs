# DBA Dash - Documentation

## Intro

This is the repository for [http://dbadash.com](http://dbadash.com), the documentation website for open-source SQL Server monitoring tool, [DBA Dash](https://github.com/trimble-oss/dba-dash).

This is a static website based on [Doks](https://getdoks.org/)

## Getting started

```
npm install
```

```
npm run dev
```
