# DBA Dash Documentation

Based on [Doks](https://getdoks.org)

## Setup

* Clone the repo
* Run `npm install`
* Run `npm run start`
* Website is available at http://localhost:1313/

## Add docs page

* npm run create docs/help/introduction/index.md
* Change draft to false
* Set parent
